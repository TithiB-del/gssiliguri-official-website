### official-website
The official website of Girlscript Siliguri! <br>
Introduced as a part of Hacktoberfest 2022, we plan to build a website for girlscript siliguri which will be open sourced. <p>
Here are some tips to start contributing, especially if it is your first time: 
<ol><li>Make sure to check out the issues tab to see if there is any issue that you can work on.</li>
<li>Make sure to read the contribution guidelines before making a PR.</li>
<li>Make sure to check out the code of conduct.</li>
<li>Make sure to check out the README file to understand the project better.</li>
<li>Make sure to check out the project board to see the progress of the project.</li>
</ol>
</p>
<p>Steps to contribute:
<ol>
<li>Star & fork this repo and make sure that you have git installed in your system </li>
<li>Clone repo locally using git clone https://github.com/girlscriptsiliguri/official-website </li>
<li>After cloning make sure you create a new branch by using git checkout -b my-branch. </li>
<li>Add your name in the CONTRIBUTORS.md file. </li>
<li>Add the modified/created files to the staging using ''' git add . ''' </li>
<li>Commit the changes made into the checked out branch using ''' git commit -m "commit message" ''' </li>. 
<li>Push the changes using git push origin my-branch. </li>
</p>
